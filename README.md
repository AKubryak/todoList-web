1. Перейдите в папку `backend/`:

```
cd backend
```

2. Запустите сервер:

```
php artisan serve
```

3. Перейдите в папку `frontend/`:

```
cd frontend
```

4. Запустите сервер разработки:

```
npm run dev
```

## Доступ к приложению

* Бэкенд: [http://localhost:8000](http://localhost:8000/)
* Фронтенд: [http://localhost:5173](http://localhost:5173/)
